问题描述
----------------
给定一个未经排序的整数数组，找到最长且连续的的递增序列。

示例
----------------
示例 1:

输入: [1,3,5,4,7]

输出: 3

解释: 最长连续递增序列是 [1,3,5], 长度为3。

尽管 [1,3,5,7] 也是升序的子序列, 但它不是连续的，因为5和7在原数组里被4隔开。 

示例 2:

输入: [2,2,2,2,2]

输出: 1

解释: 最长连续递增序列是 [2], 长度为1。

注意：数组长度不会超过10000。

C语言
-----------------------
```C
int findLengthOfLCIS(int* nums, int numsSize){
    int A=0,B=0,t=0,Max=0;
    for(int i=0;i<numsSize-1;i++)
    {
        if(nums[i]<nums[i+1]){
            B=i+1;
        }
        else{
            A=i+1;
            B=A;
        }
        if(Max<B-A+1){
            Max=B-A+1;
        }
    }
    if(numsSize==1){
        return 1;
    }
    return Max;
}
